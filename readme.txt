== Description ==

Kashiwazaki SEO Link Doctor は、WordPressサイト内のリンク切れを効率的に検出・管理するための軽量プラグインです。サイト内のすべてのページに含まれる内部・外部リンクをチェックし、404エラーやサーバーエラーなどの問題を素早く発見できます。

**主な機能:**

* **軽量なHTTPチェック** - HEADリクエストでサーバー負荷を最小限に
* **リアルタイム進捗表示** - チェック中は進捗状況（n/n件、%）をリアルタイムで表示
* **ビジュアルプレビュー** - 該当リンクをiframeでハイライト表示、自動スクロール付き
* **スマートキャッシュ** - 24時間のステータスキャッシュ、手動クリア可能
* **個別再チェック** - ページ全体を再チェックせずに個別リンクのみ再チェック
* **結果エクスポート** - クリップボードにコピーしてExcel等に貼り付け
* **組み込みドキュメント** - プラグイン内に詳細な使い方ガイドを搭載

**除外されるリンク:**

以下のリンクタイプは自動的に除外されます：
* tel: (電話番号リンク)
* mailto: (メールリンク)
* javascript: (スクリプトリンク)
* data:, ftp:, file:, sms:, skype:

== Installation ==

1. プラグインフォルダを `/wp-content/plugins/` にアップロード
2. WordPress管理画面の「プラグイン」メニューから有効化
3. 管理メニューの「Kashiwazaki SEO Link Doctor」からアクセス

== Frequently Asked Questions ==

= キャッシュはどのように動作しますか？ =

リンクステータスの結果は24時間キャッシュされます。「ステータスキャッシュをクリア」ボタンで手動クリアするか、個別/一括再チェックボタンでキャッシュをスキップできます。

= 外部リンクで403エラーが出るのはなぜですか？ =

一部のウェブサイトはボットからのアクセスを制限しています。ブラウザで直接アクセスすると正常に表示される場合があります。手動で確認してください。

= どのステータスコードが問題を示しますか？ =

* 200 = 正常（問題なし）
* 301/302 = リダイレクト（URL更新を検討）
* 403 = アクセス禁止（ボット制限の可能性）
* 404 = ページが存在しない（リンク切れ）
* 500/503 = サーバーエラー（一時的な問題）
