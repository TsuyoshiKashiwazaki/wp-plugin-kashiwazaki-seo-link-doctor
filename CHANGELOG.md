# 変更履歴

このプロジェクトの主な変更はこのファイルに記録されます。

## [1.0.0] - 2025-12-29

### 追加
- Kashiwazaki SEO Link Doctor 初回リリース
- 軽量なHEADリクエストによるHTTPステータスチェック
- リアルタイム進捗表示（n/n件、%）
- iframeによるビジュアルプレビュー（ハイライト表示、自動スクロール）
- WordPress transientsを使用した24時間ステータスキャッシュ
- 手動キャッシュクリア機能
- 個別リンク再チェックボタン（キャッシュスキップ）
- ページ全体の一括再チェック
- Excel/スプレッドシート用TSVクリップボードエクスポート
- ソート可能な結果テーブル
- 長いURLの折りたたみ/展開トグル
- 使い方ガイド付き説明書タブ
- 非HTTPスキームの自動除外（tel:, mailto:, javascript:等）
- プラグイン一覧に設定リンク追加
- アンインストール時のキャッシュ自動削除
